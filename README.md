# Сайт для клиента по услугам арбористов

**🌐 [Сайт](https://arboristy-spb.ru/)**   
**📅 Проект выполнен в сентябре 2023**  
**📦 Добавлен на GitHub в марте 2026**

## О проекте
Коммерческий заказ от компании. Разработана главная страница со всеми необходимыми блоками. Реализована адаптивная верстка под все устройства

## Реализовано
*   Адаптивная сетка на Flexbox/Grid
*   Слайдеры на Slick-Slider
*   Бургер-меню для мобильных устройств
*   Мультикнопка для связи в соц-сетях и по телефону
*   Модальное окно
*   Аккордеон для блока вопросов и ответов

## Почему код добавлен сейчас
В 2023 году при работе над проектом мы не использовали Git. Сейчас я привожу портфолио в порядок и добавляю ключевые коммерческие проекты на GitHub.

## Структура
*   `/css` — стили
*   `/js` — скрипты
*   `/images` — изображения
*   `/fonts` — шрифты
*   `index.html` — главная страница

## Технологии
HTML, CSS, JavaScript, Slick-Slider, jQuery
